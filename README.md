## Sup Boar - Tranfering figma desing into a web page.

Reference url: https://www.youtube.com/watch?v=sBvdgXjIlxM&t=11639s

### Local development setup:
 - Webpack server with Hot Module Replacement. Gives an ability to dynamically update javascript and css files for a web page. This significantly speeds up the development and instantly reflects the code changes.
 - Pug was used as a template engine.
 - SCSS preprocessor.
